# Transport accessibility of selected Katowice cultural institutions 
### Medialab Katowice, Shared Cities: Creative Momentum project

Map of transport accessibility of cultural events in Katowice in the form of a heatmap presenting travel times from all address points across the city to 14 selected institutions. Travel times by public and private transport, calculated using the Google Distance Matrix API, were visualized using isochrones referring to four categories of accessibility: convenient, standard, demanding, difficult. The model also uses data on places of residence and means of transport used to access cultural events obtained from the respondents in the study of Katowice cultural events.
